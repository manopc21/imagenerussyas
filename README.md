# Feed de fotos estilo Instagram

Este projeto exibe as imagens da pasta `your_posts` em um carrossel horizontal que troca automaticamente a cada 2 segundos.

## Como usar

1. Abra o arquivo `index.html` no navegador, ou rode o servidor local com:
   ```bash
   python server.py
   ```
2. Acesse `http://localhost:8000`.

## Para subir no GitHub

1. Crie um repositório no GitHub.
2. Envie estes arquivos:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `manifest.json`
   - `your_posts/`
3. Ative o GitHub Pages no repositório.

> Observação: para o GitHub Pages funcionar corretamente, as imagens precisam estar dentro do repositório e o projeto deve ser servido como página estática.
